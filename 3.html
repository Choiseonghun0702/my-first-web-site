JavaScript

JavaScript는 웹 페이지에 복잡한 기능을 구현할 수 있게 해주는 스크립팅 또는 프로그래밍 언어입니다. 웹 페이지는 단순히 거기에 앉아서 사용자가 볼 수 있도록 정적 정보를 표시하는 것 이상을 수행할 때마다 적시에 콘텐츠 업데이트, 대화형 지도, 애니메이션 2D/ 3D 그래픽, 스크롤 비디오 주크박스 등 — 아마도 JavaScript가 관련되어 있을 것입니다. 이것은 표준 웹 기술 레이어 케이크의 세 번째 레이어이며 그 중 두 개( HTML 및 CSS )는 학습 영역의 다른 부분에서 훨씬 더 자세히 다루었습니다.
